# WarChaos no Linux 🐧🎮

Este projeto tem como objetivo fornecer um guia otimizado e documentado para rodar o jogo **WarChaos** em distribuições Linux, utilizando o **Heroic Games Launcher**, **Wine-GE** e **DXVK**.

## 🚀 Sobre o Projeto
Muitos usuários enfrentam problemas de congelamento ao carregar partidas ou erros de inicialização do launcher devido a dependências do .NET e configurações de API de áudio no Wine. Este repositório centraliza as correções testadas e as melhores variáveis de ambiente para garantir que o jogo rode com a melhor performance possível (estável em APUs Ryzen e GPUs Dedicadas).

> Crie sua conta: https://wf.warchaos.com.br/account/register?ref=e213ec421ff4

## 🛠️ O que você encontra aqui:
- Passo a passo da criação do prefixo 64-bit.
- Lista de dependências essenciais (`vcrun2022`, `.NET 8`).
- Ajustes críticos de compatibilidade (Windows 7 mode).
- Variáveis de ambiente para eliminar stuttering.
